# Pulse Wall

Enterprise topology preview and CLUS26 DR Pulse Wall project assets.

- **Topology preview:** serve the `topology/` folder over HTTP and open `topology/preview/index.html` (see `topology/preview/README.md`).
- **Context:** see `context.md`.
